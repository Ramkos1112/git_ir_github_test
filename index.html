Git commands

  • init - creating a git repository, located in the current folder.
  • status - gives current information of the git repository
  • add - adds specified files to those that are ready to be commited (staged)
  • rm --cached filename || restore --staged - removed selected files from the staged group
  • commit "message about comitting" - creates a code version from staged files and adds a message about changes.
