Git commands

  • init - creating a git repository, located in the current folder.
  • status - gives current information of the git repository
  • add - adds specified files to those that are ready to be commited (staged)
  • rm --cached filename || restore --staged - removed selected files from the staged group
  • commit "message about comitting" - creates a code version from staged files and adds a message about changes.
  • restore - returns specified files to the previous version, so long as they have not been staged.
  • push - sends local repository commits to the outer repository
  • pull - downloads the outer repository changes to the inner repository.
  • log - shows a list of the commits
  • clone - makes a copy of the file in your device.
  • checkout commitID|branchname -
  • branch newBranchName - creates a new branch.
